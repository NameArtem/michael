# michael
A/B testing and multi-armed bandit made easy

Supports frequentist and Bayesian A/B tests as well as multi-armed bandits. See `getting_started.ipynb`.

## Installation

```
git clone https://bitbucket.org/centralonline/michael/
cd michael
pip install .
```
